# Clock of clocks

A clock made of clocks using React and Tailwindcss.

## Demo

Check out the live site here: [Clock of clocks](https://jsubroto.github.io/clock-of-clocks/)

## Installation

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

## Technologies

- React
- Tailwind CSS
- Vite
